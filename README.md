# Customer Segmentation using Flask and Machine Learning

## Project Description
This project is a customer segmentation web application built using Flask and machine learning.
Users upload a CSV file, and the system applies K-Means clustering to segment customers.
The results are displayed using visual charts.

## Features
- CSV file upload
- Data preprocessing and validation
- Customer segmentation using K-Means
- Graphical visualizations
- Flask-based web interface

## Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset Requirements
The CSV file must contain the following columns:
- Age
- Gender
- Income
- Spending Score

## How to Run the Project
1. Install dependencies:
pip install -r requirements.txt
2. Run the application:
python main.py
3. Open browser and go to:
http://localhost:5000
